Re-creation of LEGO Racers that features new kart physics, improved bots, online multiplayer and more.

I had wanted to create my own kart racer and loved playing Lego Racers when I was a kid so I set out to make a simple clone that quickly grew into a larger project that featured updated graphics, networked multiplayer, kart building & more.

The project had a post about it pinned and featured on Rock Raiders United when it was still active.
